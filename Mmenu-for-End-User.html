Mmenu for End User

Web - Mobile

Mmenu for End User is an engagement web app that provides a digital menu, process tracking and payment methods. The customers can do all the ordering process just by QR code scanning without further face-to-face interaction due to contact restriction requirement during the Covid pandemic.

Role: UX researcher, UI designer

Timeline: 4 months

Tools: Figma, Illustrator, Slack

Goals

Good user experience with a practical, easy to use interface and quick process to place order. The screens should display neat and sufficient contents to diners and merchants’s needs.

Conclusion

After trying several top food menu apps and from the local market competitors, interviewing the business owners, their staffs and customers. My team got the detailed analysis of user problems and desires:

- A clean layout and convenient category display
- Legible contents
- Easy to use
- Many dishes have long name
- Many dishes have size and extra topping
- Promotion and loyalty program

Visual design

With tight width and space between from Open Sans font, this allows more characters can be displayed on a single text area. As a result, we can display more content without worrying about breaking the layout and still highly legible.
As an impactful color in the F&B industry and also the brand color of Mmenu, Red was chosen to be primary color.

User flow

Place order

After scanning QR code, users view the menu which contains a list of dishes and categories on the left bar. Users can pick size, extra topping and add note in dish detail screen. All the selected dishes will be listed in cart. In this step, users can add coupon and enter personal information ( required or optional) before confirming order.

User flow

Track process

After placing order, users can view the process, dishes status and still can change or request to change order. Users can’t change order when dish is in “cooking” and “done” states.

User flow

Payment

There are different payment options for users after finishing the meal, which can be set by business owners in Mmenu for Admin. Users can view a digital receipt after picking a payment method.

User flow

Activity history

Activity History feature allows users to view previous order and payment details from restaurants, cafes or bar also in Mmenu system.
