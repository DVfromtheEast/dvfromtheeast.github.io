Vietrip

IOS - Android - Portrait mode

Vietrip is a system of tour packages booking managements, which can be used by both tour operator and tourist. It also offers on-going tour tracking and other services.

Role: UI/UX designer, Illustrator

Timeline: 4 months

Tools: Figma, Illustrator

Goals

Make the tour booking experience easier and more enjoyable. Beside that, this app also contains other features for Visa service, chat room and process tracking. The interface should be easy to use and well-organized with large amount of content.

Conclusion

By searching and collecting information about client’s business and trying other apps and website which have the similar business model. My team has came up with ideas and solutions that can enhance the product value. Next my team worked on wireframes before processing with visual design.

Visual design

This mobile app featured material design as the design system. The app design was not only focused to the people who search for tour packages or articles but also to those who are always on the go.
The brand colors gave the contrast its needed between elements and i have gave specific role to each color in the interface, green for CTA and pink for emphasized content.
The button components have gradient background to stands out from the surface. Users can interact more comfortably while being outside with the objects that have depth.

User flow

On boarding/ Log In

The user flow hasn’t been designed only for the existing customers, tour guide and the admin, but also for the newcomers. To welcome users and excite them about the experience ahead, the onboarding screens which contains illustrations filled with brand colors were added.

User flow

Tour booking

Home screen consists 2 tabs. The first tab - Tour Packages displays several travel pakages that might interest tourists. The packages has been categorized for customer needs and promotional purposes.

User flow

Tour packages detail

Tour packages details screen provides all the information about the trip. Beside the photos which were uploaded by the tourists or the tour guide during the trip, it displays the price, duration, detailed information of the journey from departure to return, agreement and reviews.

User flow

Travel guide

The second tab - Travel guide shows a large amount of helpful information for travel: Tourism articles, visa, about local culture, food and tips for tourists in a specific place.

User flow

Group chat

Admin, tour guide and customers can create chat groups. People in a tour can communicate with each others conveniently. It also helps tour guide better in controlling the trip.

User flow

Account - Visa service

Screen “Me” allows users to check their current and past activities, edit personal profile and other information of the service.
Screen “Visa” provides service of visa. It displays all the requirement for the submission and price in total.
